🔢 Number System Converter

📌 Overview

The Number System Converter is a simple web application that converts numbers between different number systems:

Decimal (Base 10)

Binary (Base 2)

Hexadecimal (Base 16)

This project demonstrates basic JavaScript logic + DOM manipulation with a clean UI.

🌟 Features

🔢 Convert numbers between Decimal, Binary, and Hexadecimal\

⚡ Instant conversion on button click

🛑 Input validation for incorrect values

🎨 Clean and modern UI

📱 Responsive design

🛠️ Tech Stack

HTML5

CSS3

JavaScript (Vanilla JS)
